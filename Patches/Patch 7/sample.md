# Patch 7 smart-writing sample

This sample is meant to show the **Patch 7 writing workflow**:

- type `/` in the editor to open slash commands
- select text to reveal the contextual formatting toolbar
- use typewriter mode when you want a calmer drafting experience
- use paragraph focus mode when editing a single section
- paste or drop images straight into the markdown source
- track a word goal from the bottom status bar

## Try these interactions

1. Type `/meeting` and choose the meeting notes snippet.
2. Select this sentence and try **Bold**, **Italic**, or **Note** from the selection toolbar.
3. Open **Tools** and enable **Typewriter mode**.
4. Enable **Paragraph focus** and move between the sections below.
5. Set a **word goal** and keep writing until the progress chip updates.
6. Paste or drag an image into the editor.

> [!TIP]
> Patch 7 is designed to reveal controls only when the editing context actually needs them.
