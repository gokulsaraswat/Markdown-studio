# Markdown Studio Local — Changelog

## Patch 7 — Smart writing
- added **slash commands** directly inside the editor for headings, quick snippets, media, tables, callouts, and timestamps
- added a **contextual selection toolbar** that appears only when text is selected
- added **typewriter mode** to keep the active line centered while drafting
- added **paragraph focus mode** to dim surrounding text and isolate the current block
- added a **spellcheck toggle** for browser-native spelling hints
- added **word goal tracking** with progress visible in the status bar
- added **clipboard image paste** and **image/GIF drag-drop embed** inside the editor
- upgraded the **Insert** and **Tools** panels so Patch 7 features are reachable without cluttering the main header
- refreshed the **README**, **sample markdown**, **QA checklist**, and **agent docs** for the smart-writing workflow

## Patch 6 — UX redesign
- replaced static header status labels with a **document popover** opened from the file name
- replaced the wide static action rows with **grouped menus** for format, insert, export, view, tools, panels, and workspace features
- added a **quick-format strip** so common markdown actions stay one click away
- added **divider guidance** with hover help, live split ratios while dragging, and double-click reset
- tightened the **header, pane headers, and status bar** to reclaim writing space on laptop screens
- changed the default split to an **editor-first 62 / 38** layout for better drafting space
- refreshed the **welcome guide, release notes, README, sample markdown, changelog, and agent docs** for Patch 6

## Patch 5 — Final polish
- added a unified **workspace settings** modal for writing, workspace, preview, help, and cleanup controls
- added a **quick-start guide** shown on first launch after the patch
- upgraded notifications to a **stacked toast** system with clearer success, info, warning, and error states
- added **reduced motion** support and an **editor font size** control
- added an in-app **release notes** modal covering Patch 1 through Patch 5
- added agent-facing delivery docs: **FINAL_QA_CHECKLIST.md**, **PATCH_5_AGENT_GUIDE.md**, **PATCH_5_DIFF.txt**, and **PATCH_5_MANIFEST.txt**
- refreshed the default welcome content and sample markdown for the final build
